# Kawabanga

Repositório básico do Jogo "Kawabanga" desenvolvido pelos alunos Diogo Corrêa do Nascimento e Luis Augusto Petrochewski Silva para a disciplina de Técnicas de Programação, ministrada por Jean Marcelo Simão.

O jogo tem como objetivo o aprendizado na linguagem C++ Orientada a Objetos


## Atualização 23/06/2026
